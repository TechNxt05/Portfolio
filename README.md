# Amritanshu Yadav's Portfolio

Welcome to my personal developer portfolio, built with Next.js, specialized React components, and modern web integrations.

## 🚀 Live Demo

[amritanshuyadav.vercel.app](https://amritanshuyadav.vercel.app) *(Update this link once deployed)*

## 🛠️ Built With

- **Framework:** Next.js
- **Styling:** Tailwind CSS, Framer Motion
- **Language:** TypeScript
- **Icons:** Lucide React
- **Deployment:** Vercel

## 🌟 Key Features

- **Interactive AI Chatbot (`knowledge.json`):** Integrated assistant providing contextual information about my projects, experiences, and education using tailored LLM integration.
- **Dynamic 3D Visuals & Components:** Includes interactive elements like the Chess Hierarchy for projects, multi-dimensional flip-cards for skills, and glowing timelines.
- **Responsive & Modern UI:** Designed with an immersive dark theme focusing on clarity, animations, and top-tier user experience.

## 💻 Running Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/TechNxt05/Portfolio.git
   cd Portfolio
   ```

2. **Install dependencies:**
   ```bash
   npm install
   # or yarn / pnpm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. **View the site:**
   Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
